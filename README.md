## 📜 Cheat Sheet (Preview)

```bash
# Run container  
docker run -d -p 8080:80 nginx  

# List running containers  
docker ps  

# Build custom image  
docker build -t myapp:1.0 .  

# Docker Compose  
docker-compose up -d  
docker-compose down  
```
